# Frontend Design Skill (Olea Terra)

Dieser Skill leitet die Erstellung von hochwertigen, E-Commerce-optimierten Frontend-Interfaces für die Olivenöl-Marke "Olea Terra" an. 

## Design Thinking für Olea Terra

- **Zweck**: Verkauf von Premium-Olivenöl, Hervorheben von Qualität (Laborgeprüft), B2B-Leads generieren.
- **Ton**: Organisch, luxuriös, natürlich, erdig, raffiniert. Keine generischen Tech-Vibes. Vertrauen aufbauen.
- **Brand Colors**: 
  - `olive`: (950 für starken Kontrast, 50-100 für softe Backgrounds)
  - `earth-green`: (Primary Actions)
  - `harvest-gold`: (Highlights, Sterne, Badges)
  - `terracotta`: (Akzente, Tags)
  - `charcoal`: (Typografie, Headings)

## Frontend Ästhetik Guidelines

### Typografie
- Nutze Serif-Fonts (`font-serif`) für Headings (`h1`, `h2`, `h3`) um Eleganz und Tradition auszustrahlen.
- Moderne Sans-Serif Fonts für Fließtexte und UI-Elemente (`Buttons`, `Labels`).

### UI Layout & Conversion
- **Sticky Add-to-Cart**: Auf Produktdetailseiten immer sichtbare Kauf-Optionen.
- **Trust Elements**: Symbole wie `ShieldCheck` (Laborgeprüft), `Truck` (Schneller Versand), `Leaf` (Bio) strategisch platzieren.
- **Whitespace**: Großzügig nutzen, um das Produkt atmen zu lassen und Premium-Wirkung zu verstärken.
- Formulare (wie B2B) müssen weiche Borders (`rounded-xl` oder `rounded-3xl` für Container) haben, keine harten Kanten.

### Motion & Animation
- Fließende, weiche Animationen (`duration-300`, `duration-500`).
- Bilder auf Scroll leicht skalieren (`scale-105` zu `scale-100`) für einen Parallax-Feel.
- Keine aggressiven Bounces, eher langsame Fade-ins.

## ❌ NIEMALS verwenden
- Harte Drop-Shadows (nutze weiche, große Shadows wie `shadow-xl` oder `shadow-2xl`).
- Generische Blueprint-Farben (Bright Blue, Standard Red). Errors sollten in abgetönten Rot-Tönen oder minimalistisch dargestellt werden.
- Abrupte Layout-Wechsel ohne Animation.
